# Customer Data Wrangling Project

## Objective
To perform data cleaning and transformation on a customer dataset and prepare it for analysis.

## Tasks Performed
- Data Familiarization
- Data Quality Assessment
- Missing Value Handling
- Duplicate Removal
- Text Standardization
- Date Format Standardization
- Feature Engineering

## Tools Used
- Python
- Pandas
- VS Code

## Output Files
- customer_data.csv
- customer_cleaned.csv
- data_cleaning.py
- Data_Dictionary.xlsx
- Data_Quality_Report.docx

## Feature Engineering
Created a new column:
- Customer_Age

## Result
Generated a clean and analysis-ready dataset.
